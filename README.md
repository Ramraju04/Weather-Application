🌦️ Weather Dashboard

A modern Weather Dashboard built using HTML, CSS, and JavaScript, powered by the OpenWeatherMap API.
This app provides real-time weather updates, a 3-day forecast, and detailed weather highlights such as temperature, humidity, wind speed, pressure, UV index, sunrise, and sunset times.

🚀 Features

🔍 Search by City – Get weather details for any location.

🌡️ Temperature Conversion – Toggle between Celsius and Fahrenheit.

☀️ Current Weather – Displays temperature, feels-like value, description, and weather icon.

📊 Weather Highlights – Humidity, wind speed, pressure, cloudiness, UV index, sunrise, and sunset.

📅 3-Day Forecast – Shows max/min temperature, icons, and descriptions for the next 3 days.

⏱️ Auto Update – Displays the last updated time dynamically.

📱 Responsive Design – Optimized for desktop and mobile devices.

------

🛠️ Tech Stack

Frontend: HTML, CSS (Poppins font, FontAwesome icons)

JavaScript: Fetch API for API calls, DOM manipulation

API: OpenWeatherMap API
